Online Library application that allows CRUD operations of Books, Authors, Categories & Publishers. Uses an in-memory data-base. Combines both Thymeleaf & H2.

Technologies used:
- IntelliJ Idea
- Spring(boot)
- Maven
- POSTMAN
- H2 Database
- Thymeleaf
- Lombok
- JPA (Java Persistence API)

Dependencies:
- SpringWeb
- Thymeleaf
- Lombok
- Spring DataJPA
